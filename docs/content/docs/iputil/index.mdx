# IP 工具包 (iputil)

`iputil` 是专为 Web 应用设计的 IP 地址处理工具包，提供了获取客户端真实 IP、验证 IP 格式等常用功能。

## 🎯 设计目标

- **简单易用**: 函数接口简洁明了，开箱即用
- **高度兼容**: 支持各种代理和 CDN 场景
- **零依赖**: 基于 Go 标准库实现，无外部依赖
- **性能优先**: 高效的实现，适用于高并发场景

## 📦 安装

```bash
go get github.com/woodchen-ink/go-web-utils/iputil
```


## 🧪 测试

```bash
# 运行测试
go test github.com/woodchen-ink/go-web-utils/iputil

# 查看覆盖率
go test github.com/woodchen-ink/go-web-utils/iputil -cover
```
